# Bell Performance 7.0.9 — Mission Intelligence

Bell Performance 7.0.9 expands First Flight with a mission-first planning system.

## First Flight mission paths

### Specific Event
The athlete can select a dated event, including HYROX, CrossFit, powerlifting, strongman, combat sports, road races, triathlon, obstacle racing, Tactical Games, fitness tests, or a custom sport event.

The app derives:
- Strength priority
- Engine priority
- Recommended Strength and Engine frequency
- Total block duration from the selected start date to the event
- Foundation, build, event-specific, peak, taper, and event-week phases
- Event-specific conditioning labels and prescriptions

### Performance or Body Composition
For athletes without a competition date, First Flight offers:
- Fat Loss
- Muscle Building
- Body Recomposition
- Mobility & Longevity
- Sport-Specific Performance
- General Hybrid Fitness
- Strength Development
- Endurance Development

These goals create repeatable development blocks with deloads and end-of-block reviews.

## Existing functionality preserved
This build retains the 7.0.x onboarding tour, movement limitations, workout-location equipment substitutions, engine result logging, history reviews, milestones, adaptive deloads, missed-session management, Dual Mission coordination, and first-block launch controls.

## Deployment
Upload the contents of this folder to the root of a GitHub Pages repository. A service worker caches the local application assets for installable/offline use.
